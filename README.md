# Tip
1. 질문. 벤더프리픽스를 해줘야 할 때 스크립트로 적용하면 어떻게 작업을 해줘야 하나요?
   - 바닐라 스크립트로 적용할 땐 아래 3개 방식으로 처리해줘야 한다.
   - elem.style.transform
   - elem.style.webkitTransform
   - elem.style.mozTransform
   - 위 3개 방식 외엔 자동으로 해주는 autoprefixer 같은 라이브러리 사용
